Welcome to the Word Highlighter Plugin Guide

This plugin allows you to highlight specific words on a webpage with customizable colors. Here’s how it works:

Features:

  • Double-click to highlight: When you double-click a word on the page, it will be highlighted with a unique background color. If the word has already been highlighted, it will retain its color.
  • Triple-click to change the color: If you triple-click a highlighted word, its color will change to a new randomly generated background color.
  • Right-click to clear all highlights: If you right-click anywhere on the page, all highlights will be removed, and the word color map will be reset.

How it works:

Once the plugin is installed, it will automatically begin listening for user interactions on the webpage.

1. Double-clicking on a word

When you double-click on a word, the plugin highlights it with a new random background color. If the word has been highlighted before, the same color is used for consistency.

For example, if you double-click on the word "highlight", the word will be highlighted with a new color.

2. Triple-clicking to change color

If you triple-click on a previously highlighted word, the plugin will generate a new random color and change the highlight color of that word.

For example, if the word "highlight" was already highlighted in yellow, triple-clicking it will change the highlight color to a new color.

3. Right-clicking to clear highlights

When you right-click anywhere on the page, all highlights will be cleared, and the plugin will reset the word color map, allowing you to start fresh.

Additional Information:

This plugin ensures accessibility by adjusting the text color to maintain proper contrast against the background highlight color. It generates a suitable text color (either black or white) based on the background color brightness, ensuring readability.
